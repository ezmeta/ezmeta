# EZ Meta Demo Quick Start Guide

## 5-Minute Setup

Follow these steps to quickly prepare for recording the EZ Meta demo:

1. **Start the development server**
   ```bash
   npm run dev
   ```

2. **Open the application in your browser**
   - Navigate to `http://localhost:3000`

3. **Prepare your recording software**
   - Set to record at 1080p resolution
   - Configure to capture system audio if narrating
   - Test microphone if recording with voice

## Demo Path

Follow this sequence for a smooth demo flow:

1. **Landing Page** (10 seconds)
   - URL: `http://localhost:3000/`
   - Show: Main hero section and features

2. **Dashboard** (20 seconds)
   - URL: `http://localhost:3000/dashboard`
   - Action: Click "Sync" button
   - Show: Metrics and graphs

3. **Meta Ads Page** (20 seconds)
   - URL: `http://localhost:3000/meta-ads`
   - Action: Find "Insurans Kereta Promo" campaign
   - Action: Click "Generate Creative" button on that row

4. **Creative Studio Modal** (20 seconds)
   - Action: View analysis
   - Action: Click through tabs (Copywriting, Visual Ideas, Video Script)
   - Show: Content in each tab

5. **Pricing Page** (10 seconds)
   - URL: `http://localhost:3000/pricing`
   - Action: Hover over "Get Started Free" button

## Troubleshooting

- If the Creative Studio modal doesn't appear, try refreshing the page and clicking the button again
- If the dashboard doesn't load, check the console for errors and restart the development server if needed
