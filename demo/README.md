# EZ Meta Demo Recording Instructions

## Overview

This document provides instructions for recording the EZ Meta demo video for Meta Review & Marketing purposes. The demo should showcase the key features of EZ Meta, focusing on its ability to optimize Facebook & Instagram ads using AI.

## Prerequisites

1. A clean browser window without unnecessary tabs or bookmarks
2. Screen recording software (OBS Studio, Camtasia, or similar)
3. 1080p resolution minimum
4. Stable internet connection

## Setup Before Recording

1. Make sure the development server is running:
   ```
   npm run dev
   ```

2. Clear browser cache and cookies for a fresh session
3. Set browser zoom to 100%
4. Have the demo script open in a separate window for reference

## Recording Flow

Follow the detailed script in `VIDEO_DEMO_SCRIPT.md`. Here's a summary:

### 1. Landing Page (00:00 - 00:10)
- Start at the landing page
- Scroll down slightly to show the main features
- Script: "Memperkenalkan EZ Meta, pembantu AI yang mengoptimumkan iklan Facebook & Instagram anda berdasarkan data sebenar."

### 2. Dashboard (00:10 - 00:30)
- Navigate to the Dashboard
- Show the graphs and metrics
- Click the "Sync" button to demonstrate data refreshing
- Script: "EZ Meta menghubungkan akaun iklan anda dan memantau metrik kritikal seperti Spend, CTR, dan CPC dalam satu dashboard yang bersih."

### 3. AI Creative Generation (00:30 - 00:50)
- Navigate to Meta Ads page
- Find the "Insurans Kereta Promo" campaign with low CTR (0.8%)
- Click the "🪄 Generate Creative" button
- Wait for the modal to appear with analysis
- Script: "Apabila iklan anda tidak berprestasi, AI Creative Studio kami akan menganalisis puncanya dan menjana copywriting serta idea visual baru untuk meningkatkan ROI anda."

### 4. Multi-format Content (00:50 - 01:10)
- Click through the tabs in the Creative Studio modal:
  * Copywriting
  * Visual Ideas
  * Video Script
- Pause briefly on each tab to show the content
- Script: "Dapatkan segalanya dari headlines hingga ke skrip video pendek yang direka khusus untuk audiens anda."

### 5. Call to Action (01:10 - End)
- Navigate to the Pricing page
- Hover over the "Get Started Free" button
- Script: "Kurangkan kos, tingkatkan jualan. Mula secara percuma dengan EZ Meta hari ini."

## Technical Notes

1. The demo uses mock data for demonstration purposes
2. The "Insurans Kereta Promo" campaign has been specifically configured with a low CTR (0.8%)
3. The AI Creative Studio has been pre-configured with Malaysian language content
4. All pages have been optimized for the demo

## Post-Recording

1. Review the recording for any issues
2. Trim the beginning and end if necessary
3. Add any required branding elements
4. Export in high quality (1080p minimum)

If you encounter any issues during the recording process, please contact the development team.
