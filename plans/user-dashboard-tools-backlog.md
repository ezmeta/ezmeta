# User Dashboard Tools Backlog

Status: Saved from product discussion for next implementation phase.

## Core Killer Features (High Priority)

1. **AI Ad Copywriter**
   - Generate ad copy in Hard Sell / Soft Sell / Storytelling modes.
   - Localized tone for Malaysia (BM/EN/Manglish-safe variants).
   - Output bundles: headline, primary text, CTA.

2. **AI Smart Pilot**
   - Daily optimization suggestions (budget/audience/stop-loss).
   - Explainable reasons for each recommendation.
   - Quick actions: Accept / Ignore / Snooze.

3. **Telegram AI Alert**
   - Scheduled summaries (e.g., 3x/day).
   - Critical anomaly alerts (spend spikes, ROAS drop, CTR drop).
   - Noise control modes (Critical only / Full feed).

4. **Winning Ad Detector**
   - Rank top ads by efficiency signals (ROAS, CPA, CTR, CPP).
   - Highlight “scale candidates” with confidence score.
   - One-click action proposals (scale +20%, duplicate test set).

5. **Creative Fatigue Alert**
   - Detect fatigue from trend signals (frequency↑, CTR↓, CPA↑).
   - Alert severity tiers (info/warn/critical).
   - Auto-refresh brief suggestions (new angle/hook/visual direction).

---

## Additional Recommended Tools (Next Wave)

6. **Winning Pattern Analyzer**
   - Analyze patterns behind winning ads (hook, offer, CTA, format).
   - Focus on reusable pattern insights (not raw copying).

7. **Creative Benchmark Score**
   - Compare account performance versus niche benchmarks.
   - Surface where campaigns are above/below benchmark.

8. **Why This Works (AI Explain)**
   - Human-readable diagnosis for ad success/failure.
   - Tie explanations to measurable metrics and trend windows.

9. **Experiment Tracker (A/B Mission Log)**
   - Log hypotheses, variants, outcomes, and winner memory.
   - Prevent repeated failed tests and improve iteration speed.

10. **Daily Command Brief**
    - Single-screen morning brief: what changed, what broke, what to do today.
    - Prioritize top 3 highest-impact actions.
